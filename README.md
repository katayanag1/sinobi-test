# Sinobi-Test — Лендинг HAVAL

### Общие
-  Полноценная лендинг-страница дилерского центра
-  Адаптивная вёрстка
-  SVG-спрайты для всех иконок
-  БЭМ-именование классов

### Шапка (header)
-  Логотип (SVG из спрайта)
-  Номер телефона
-  Адрес и режим работы

### Навигация (breadcrumbs)
-  Модельный ряд HAVAL (M6, DARGO, JOLION, F7x, F7, H9, GWM Poer, GWM Poer KINGKONG)
-  Ссылки на разделы: ПОДОБРАТЬ КОМПЛЕКТАЦИЮ, ТЕСТ-ДРАЙВ, КОНТАКТЫ

### Hero-блок
-  Заголовок с акцентом на акцию "Чёрная пятница"
-  Кнопка "Зафиксировать условия"

### Таймер (timer)
-  Текст "Оставьте заявку и зафиксируйте условия"
-  Таймер обратного отсчёта (Sinoby Timer)
-  Кнопка "Зафиксировать условия"

### Карточки товаров (cards)
-  Слайдер Swiper для переключения цветов авто
-  Цветовые индикаторы с привязкой к слайдеру
-  Галерея Fancybox (увеличение фото по клику)
-  Список преимуществ с иконками из спрайта
-  3 кнопки: УЗНАТЬ СТОИМОСТЬ, РАССЧИТАТЬ КРЕДИТ, ПОДОБРАТЬ КОМПЛЕКТАЦИЮ

### Модальное окно (modal)
-  Всплывает при клике на любую кнопку
-  Поле ввода телефона с маской `+7 (___) ___-__-__`
-  Валидация номера телефона
-  Сообщение об успешной отправке
-  Закрытие: крестик, клик на фон, клавиша ESC

### Интерактив
-  Ховеры на всех кнопках и ссылках
-  Анимация кружков в списке преимуществ
-  Активное состояние цветовых индикаторов
